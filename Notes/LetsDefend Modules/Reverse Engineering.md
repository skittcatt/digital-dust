## Basics of Reverse Engineering

It is the analysis that helps security teams understand what is happening in the operations and in the programs as they begin.
- It reveals the attack methods that malware does against a system/device/etc.
- Allows the organization/company to take the necessary precautions to protect against attacks.

### Tools to be used in Reverse Engineering Lab
- **File**
	- Simple command to gain information on the original format of the file (on Linux).
- **Objdump**
	- One of the tools that can be used for this purpose on the Linux command line (BASH).
- **GCC**
	- Used to compile source C code on the Linux command line (BASH).
- **GNU Debugger (GDB)**
	- Allows to analyze files dynamically while they run on the Linux command line (BASH).
